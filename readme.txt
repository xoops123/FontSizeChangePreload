 フォントサイズを拡大するプリロード
バージョン: 1.00 掲載日: 2012/12/27

====================
説明
====================
その名のとおり、フォントサイズを拡大するプリロードです。
解凍して出来上がったhtmlディレクトリにあるcommon/FontSizeChange ディレクトリ以下のファイル群とpreload/FontSizeChange.class.php ファイルをサイトにアップしてください。

上手く動作すると、ウェブ画面の右上に「zoom」「default」と表示されたオレンジ色の丸いブロックが表示されます。

「zoom」をクリックするとページ内の文字を拡大、「default」をクリックすると元に戻ります。（ただし、ページ内でピクセルにて固定サイズを指定している場合は、文字拡大が出来ない場合があります。）

また、表示が邪魔なときは、オレンジ色の丸の下部あたりをマウスで掴むと、移動することができます。
